Enter station and date to get temperature on that particular day.

Get all data for a particular station

URL Format: http://127.0.0.1:8080/api/v1/station

URL Example: http://127.0.0.1:8080/api/v1/10

Get data for a particular station on a particular date

URL Format: http://127.0.0.1:8080/api/v1/station/date

URL Example: http://127.0.0.1:8080/api/v1/10/17560101

Get data for a particular station for a particular year

URL Format: http://127.0.0.1:8080/api/v1/yearly/station/year

URL Example: http://127.0.0.1:8080/api/v1/yearly/10/2000
